Diferentes formatos de texto

El objetivo de este sitio es mostrar los diferentes formatos te texto que pueden aplicarse con algunas etiquetas basicas de HTML para estos fines

A continuación los ejemplos:

  • Un texto en negrita es utilizado para resaltar alguna frase o palabra
  • Las cursivas son utilizadas para citar frases
  • Tambien contamos con resaltadores de texto para que lo mas importante sobresalga
  • Podemos asignar un tamaño de fuente mas pequeño del estandar
  • Se pueden tachar los textos para indicar que seran sustituidos
  • El subrayado de tambien esta presente dentro de las etiquetas disponibles
